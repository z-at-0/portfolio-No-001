# portfolio-No-001: Original Portfolio Site

## 概要

このサイトは、自分の活動・経歴・制作実績を紹介するポートフォリオサイトとして作成しました。  
スキルをわかりやすくまとめ、閲覧者に自分の強みを伝えられるよう設計しています。

## 主な内容

Works: 制作実績の紹介
About: 実装スキルや対応可能なコーディング能力の可視化

## 使用技術

使用言語: HTML5, CSS3, JavaScript (Vanilla JS)
技術要素: メディアクエリを用いたレスポンシブ対応
CSSによるレイアウト設計の基礎
ライブラリ: なし（標準技術のみで構築）

## 設計方針

情報の整理を徹底して設計しました。あえてライブラリやフレームワークを使わず、素のHTML/CSSでどこまで美しく、かつ保守しやすいコードが書けるかをテーマに構築しました。

## 学習・AI活用について

本プロジェクトは2021年に制作されたものであり、AIツール等は一切使用していません。

■ 補足

004〜006におけるAI活用は、この001〜003で培った「自力でゼロから構築する力」があるからこそ、効率的かつ安全に行えています。私の技術的なバックボーンを示す、自身のエンジニアキャリアのスタート地点といえる作品です。
